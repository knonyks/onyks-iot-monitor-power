## ESP32 power meter

Based on:

* ACS712 - 30A
* 2ma - 2ma voltage measuring transformer
* ADS1115 - I2C ADC 16bit

Programer:

* [ESP-prog](https://docs.espressif.com/projects/esp-iot-solution/en/latest/hw-reference/ESP-Prog_guide.html)
